  • TRB3

概述

  • TRB3温湿压传感器是一款集成型，简单数字信号（SDI-12），适用全天候的一款传感器。可输出：空气温度，相对湿度，大气压力指标

参数

  • 供电电压：7V～28VDC
  • 工作温度：-40℃～+70℃
  • 温度
    • 测量范围：-40°C至 +70°C
    • 测量精度
      • ±0.2°C至±0.3°C（-40°C至 +70°C）
      • ±0.1°C至±0.3°C（+20°C至 +60°C）
    • 国家气象计量站标定
      • 校准证书編号 :GQJ(C)WD2021-0545号
      • 校准设备：标准铂电阻阻温度计
        • 测量范围：(-189～+156)摄氏度
        • 准确度等级：一等标准
      • 校准环境
        • 温度：25° C
        • 湿度：16%RH
        • 气压：998hPa

  • 湿度
    • 测量范围：0-100%RH
    • 测量精度：
      • ±1.5%～±2%（0至80% RH）@25℃
      • ±2%～±3%（80%至100% RH）@25℃

        • 测量范围：(-35～+50)(露点温度)
        • 准确度等级：±0.2摄氏度(露点温度)

        • 温度：24° C
        • 湿度：12%RH
