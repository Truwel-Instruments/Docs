  • PSL8

概述

  • PSL8是一个八通道频率测量模块，可以测量低频交流频率信号、方波频率信号和开关计数。通过Modbus方式将数据可以传输到Campbell公司的CR300系列、CR1000X系列、CR6系列以及DL3000,DL1000等数据采集器

技术参数

  • 工作电压：9.6～16 Vdc
  • 电流消耗：活动 13～20mA ；静止400μA
  • 工作温度范围：-40～80 ℃
  • 方波频率信号输入范围：1Hz – 35KHz
