使用说明

参数
电源 3.6V锂电池(不可充电)，电量耗尽后可提供免费更换服务
压力精度 ±0.15％满量程
温度测量精度 ±0.2℃
水深测量范围 0~10米
压力范围 0~200千帕
超压保护 2倍压力范围
温度测量范围 -30~80℃
功耗 待机<10μA@3.3V；工作<3mA
响应时间 ＜1秒
输出接口 Type-C USB
内部数据收集内存 32MB
电池寿命 >6年（记录间隔为一小时一次）
直径 2.8cm（直径）×18cm（长度）
重量 394g
防护等级 IP68
存储
  • 32M字节
    • 每个浮点数占4字节，每一条数据包括时间、记录号、温度、电池电压、压力(千帕)、水深(米)，那么每条数据字节数等于24字节(6×4)。假设每10分钟测量并存储一次，则可存储超过13年。存满之后，设备自动循环存储，新的数据会覆盖旧的数据
USB通讯
  • 采用USB-Type-c接口，连接后，电脑端将生成一个串口，可通过电脑设备管理器查看该串口(Silicon Labs CP210x USB to UART Bridge)，如果没有安装过本系列的USB驱动，使用前需要先安装CP210x驱动
  • PC端软件 DLogger数据采集中心 ，具有时间同步、参数获取与配置、本地数据下载、远程数据下载等功能
  • 遇到问题时，用串口调试助手(波特率 921600)，发送"-debug on"(勾选回车换行)，可打开调试信息输出，通过串口输出信息判断具体原因
